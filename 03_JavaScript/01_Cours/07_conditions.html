Les conditions en JavaScript


Les opérateurs de comparaisons pour les structures conditionnelles

  • - > strictement supérieur
  • - < strictement inférieur
  • - >= supérieur ou égal
  • - <= inférieur ou égal
  • - == pour faire une comparaison de valeur -> il permet de vérifier que les valeurs de 2 variables sont identiques. Résultat boolean
  • - === pour faire une comparaison de valeur ET de type -> signifie "strictement égal à" (valeur ET type). Résultat boolean
  • - != différent de (en valeur)
  • - !== STRICTEMENT différent de (en valeur et en type)

Les opérateurs logiques

  • - AND -> &&
  • - OR -> || (altgr + 6)
  • - NOT -> !

Opérateur AND (&&)

Opérateur AND: il évalue le résultat en True si les expressions sont toutes True. Si l'une des expressions testés est False, alors le résultat sera False

  document.write(true && true);
        


  document.write(true && false);
        


  document.write(false && false);

Opérateur OR (||)

Opérateur OR : il évalue le résultat en True si une ou deux expressions sont True. Si les deux expressions sont False, alors le résultat sera False

  document.write(true || true);
        


  document.write(true || false);
        


  document.write(false || false);

Opérateur NOT (!)

Opérateur NOT : il transforme ce qui est True en False et ce qui est False en True

  document.write(!true);
        


  document.write(!false);

Structures conditionnelles if, else if, else

syntaxe

if

  if (condition){
    /* Le code si la condition est VRAIE */
  }

if - else

  if (condition){
    /* Le code si la condition est VRAIE */
  }
  else{
    /* Le code si la condition est FAUSSE */
  }

if - else if - else

  if (condition){
    /* Le code si la condition est VRAIE */
  }
  else if (condition){
    /* Le code si la condition est VRAIE */
  }
  else{
    /* Le code si toute les conditions sont FAUSSES */
  }

La condition ternaire

La condition ternaire nous permet de faire une vérification mais sur une seule ligne


Ecriture en structure conditionnelle if - else :

  let meteo = ;

  if (meteo == "pluie"){
    document.write("Tu vas être mouillé !");
  }
  else{
    document.write("Sors la moto !!");
  }


La même structure conditionnelle en écriture ternaire :

  meteo =  ;

  let ternaire = (meteo == "pluie") ? "Tu vas être mouillé !" : "Sors la moto !!";

écriture ternaire :

  • -> () = condition -> ici, est-ce que la variable meteo est égale à pluie
  • -> ? = if -> si oui, affiche "Tu vas être mouillé !"
  • -> : = else -> si non, affiche "Sors la moto !!"
