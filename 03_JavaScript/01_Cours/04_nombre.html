LES NOMBRES ET LES CALCULS

Pour initialiser un nombre, il suffit de l'écrire sans guillemets
const A = 5 * 10;
const B = 10 / 2;
const C = 15 - 5;
const D = 20 + 10;  
Nous pouvons effectuer tout type de calcul

Addition

const ADDITION = A + B;  // 50 + 5
console.log(ADDITION);
document.write("Le résultat de l'addition de A + B est égale à: " + ADDITION);
On concatène avec une chaine de caractère en utilisant le signe "+"

Soustraction

const SOUSTRACTION = C - 4;
document.write("Le résultat de la soustraction de C - 4 est égale à: " + SOUSTRACTION);

Multiplication

const MULTIPLICATION = D * 5;
document.write("Le résultat de la multiplication de D * 5 est égale à: " + MULTIPLICATION);

Division

const DIVISION = D / 10;
document.write("Le résultat de la division de D / 10 est égale à: " + DIVISION);

Modulo

const MODULO = B % 10;
document.write("Le résultat du modulo de B divisé par 10 est égale à: " + MODULO);

Raccourcis

let g = 10;

1ère façon

g = g + 5;
document.write(g);

2ème façon de faire

g += 5;
document.write(g);
