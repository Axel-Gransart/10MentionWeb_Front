LES NOMBRES ET LES CALCULS

  Pour initialiser un nombre, il suffit de l'écrire sans guillemets
  const A = 5 * 10;
  const B = 10 / 2;
  const C = 15 - 5;
  const D = 20 + 10;  
  Nous pouvons effectuer tout type de calcul

Addition

  const ADDITION = A + B;  // 50 + 5
  console.log(ADDITION);
  
On concatène avec une chaine de caractère en utilisant le signe "+"

  document.write("Le résultat de l'addition de A + B est égale à: " + ADDITION);

Soustraction

  const SOUSTRACTION = C - 4;
  document.write("Le résultat de la soustraction de C - 4 est égale à: " + SOUSTRACTION);

Multiplication

  const MULTIPLICATION = D * 5;
  document.write("Le résultat de la multiplication de D * 5 est égale à: " + MULTIPLICATION);

Division

  const DIVISION = D / 10;
  document.write("Le résultat de la division de D / 10 est égale à: " + DIVISION);

Modulo

  const MODULO = B % 10;
  document.write("Le résultat du modulo de B divisé par 10 est égale à: " + MODULO);

Raccourcis

  let g = 10;

1ère façon

  g = g + 5;
  document.write("g est égale à: " + g);

2ème façon de faire

  g += 5;
  document.write("g est égale à: " + g);

Incrémentation et décrémentation

Nous pouvons ajouter 1 ou enlever 1 avec ++ ou --

  let h = 50;
   h ++;
  document.write("h est égale à: " + h);
        

  h--;
  document.write("h est égale à: " + h);

i++ affecte puis incrémente la valeur. Dans l'exemple ci dessous, i++ incrémentera la valeur de i, mais retournera la valeur d'origine avant d'être incrémenté.

  let i = 1;
  document.write("i est égale à: " + i);
        


  let j = i ++;
  document.write("i est égale à: " + i);
  document.write("j est égale à: " + j);

++k incrémente puis affecte la valeur. Dans l'exemple ci dessous, ++k incrémentera la valeur de K, puis retournera la valeur incrémenté

  let k = 1;
  document.write("k est égale à: " + k);
        


  let l = ++ k;
  document.write("k est égale à: " + k);
  document.write("l est égale à: " + l);
