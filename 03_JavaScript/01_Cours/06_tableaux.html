Les tableaux en JavaScript

Les arrays sont extrêmement utilisés en langage de programmation, et sont très puissants

Comme une variable, ils servent à stocker n'importe quel type de données mais en quantité illimité

Les Tableaux indéxés

Déclarer un tableau indéxé

  let monTableau = [];   // 1ère façon de déclarer un tableau
  console.log(monTableau);
  
  Visible dans la console :
  []

On crée une nouvelle variable appelée myArray et on lui assigne une nouvelle instance de la classe Array.

  let myArray = new Array; // 2ème façon de déclarer un tableau
  console.log(myArray);
  
  Visible dans la console :
  []

En JavaScript, Array est une classe prédifinie qui représente une collection ordonnée d'éléments.
Lorsque vous utilisez new Array, vous créez un nouvel objet de type tableau (array).
monTableau et myArray sont des tableaux qui peuvent contenir plusieurs informations.

Affecter des valeurs au tableau, plusieurs méthodes:


1ère façon de faire
  monTableau = ["Paul", "Pierre", "Henri"];  // Chaque élément dans un tableau est séparé par une virgule 
  console.log(monTableau);
  Visible dans la console :
  (3) ['Paul', 'Pierre', 'Henri']

  console.table(monTableau);
  Visible dans la console :
    (index)    Value
      0	'Paul'
      1	'Pierre'
      2	'Henri'

La fonction .table() permet de visualiser dans la console les éléments du tableau en lien avec leurs index.


2ème façon de faire

ATTENTION: un tableau commence TOUJOURS à l'index 0 !!!!!!!

  myArray[0] = "Patrick";
  myArray[1] = "Jacques";
  myArray[2] = "Sylvie";
  myArray[3] = "Hélène";

  console.log(myArray);
  Visible dans la console :
  (4) ['Patrick', 'Jacques', 'Sylvie', 'Hélène']

  console.table(myArray);
  Visible dans la console :
  (index)   Value
    0	    'Patrick'
    1	    'Jacques'
    2	    'Sylvie'
    3	    'Hélène'

Pour sélectionner un élément de notre tableau

il suffit d'appeler :
nom-du-tableau-visé[indexDésiré]

Exemple : 

  console.log(myArray[2]);

  const PRENOM = "Georges";
