Les types de variables en JavaScript

Les différents types de valeurs

Il existe plusieurs types de valeurs que l'on peut assigner à nos variables:

  • -> Les chaines de caractères (string) => phrase, mot ou écriture entre guillemets simple (simple quotes) ou double (double quotes)
  • -> Les nombres (number) => 5, 10, 18.5
  • -> booléen (boolean) => 2 valeurs seulement True ou False
  • -> tableau (array) => ['Paul', 10]
  • -> null
  • -> classes (class)
  • -> objets (object)
  • -> fonction (function)

string

  let variable1 = "test";
  let monTexte = "Voici une phrase écrite et stockée dans une variable";

number

  let chiffre = 55;
  let negatif = -55;
  let decimal = 5.5;

boolean

  let choix = true;
  let choix1 = false;
  console.log(choix);

  Visible dans la console :

  true

  console.log(choix1);

  Visible dans la console :

  false

Afin de vérifier le type d'une variable, on utilise typeof()

  let nbr1 = 10;
  let nbr2 = "10";
  console.log(typeof(nbr1));

  Visible dans la console :
  number

  console.log(typeof(nbr2));

  Visible dans la console :
  string
