Les string en JavaScript

Nous pouvons initialiser une chaine de caratère de plusieurs façons

Avec des guillemets simples (simple quotes) :

  const PRENOM = 'John';

Avec des guillemets doubles (double quotes) :

  const NOM = "DOE";

Avec les simples quotes et un backslash (ou antislash) : Permet d'échapper un caractère dans la string

  const ADRESSE = 'J\'habite à NewYork';
  // On peut aussi noter "J'habite à NewYork"
  
  document.write("Je m'appelle " + PRENOM + " " + NOM + ", " + ADRESSE);
      

  const PRENOM2 = 'Jane', NOM2 = "Doe", LIEN = "sa femme"; 
  
  document.write("Je m'appelle " + PRENOM2 + " " + NOM2 + ", je suis " + LIEN);

Avec les backquotes (``) (altGr + 7)

Les backquotes vous permettent d'inserer du JS dans une string

  let bienvenue = `Bonjour, vous êtes les bienvenue: ${PRENOM2} et ${PRENOM}`;
  
  document.write(bienvenue);

Le " = " au niveau des strings

Contrairement au " = " d'affectation qui remplace la valeur de la variable, j'utilise ici += qui va ajouter la nouvelle valeur à la valeur déjà existante de la variable

  let fruits = "Pommes";
  document.write(fruits);

La variable fruits contient 'Pommes'

  fruits += "Fraises";
  fruits += "Mangues";
  document.write(fruits);

Maintenant, la variable fruits contient 'PommesFraisesMangues'

convertir une string vers number et inversement

Nous allons déclarer une variable avec une valeur en chaine de caractère "24"

Pour voir le type de la valeur d'une variable, on utilise typeof()

  let unNombre = "24";
  document.write(typeof(unNombre));

Pour convertir le string "24" en nombre 24

La variable unNombre est de type string. Pour la convertir en type number, nous allons utiliser la fonction parseFloat()

  unNombre = parseFloat(unNombre);
  document.write(typeof(unNombre));

Nous voyons que la variable unNombre est maintenant de type number

Convertir un number en string

Maintenant, nous voulons convertir la variable unNombre en string. Pour cela, nous allons utiliser la fonction toString()

  unNombre = unNombre.toString();
  document.write(typeof(unNombre));

Nous voyons que la variable unNombre est maintenant de type string

Convertir une string en number avec parseFloat()

Comme vu plus haut, on va utiliser la fonction parseFloat()

  let chiffreDecimal = parseFloat(chiffre);
  console.log(typeof chiffreDecimal);

  Visible dans la console :
  

  console.log(chiffreDecimal);

  Visible dans la console :

Après contrôle avec des console.log(), on voit que la variable chiffreDecimal est de type number et a une valeur de 12.5

Convertir une string en number avec parseInt()

Cette fois, on va utiliser une autre fonction appelée parseInt()

  let chiffreEntier = parseInt(chiffre);
  console.log(typeof chiffreEntier);

  Visible dans la console :
  
  
  console.log(chiffreEntier);

  Visible dans la console :

Après contrôle avec des console.log(), on voit que la variable chiffreEntier est de type number et a une valeur de 12

On voit donc que les fonctions parseFloat() & parseInt() servent toutes les 2 à convertir un type string en type number mais ne donne pas le même résultat.

parseFloat() permet un changement de string en number et prend tout les type de nombre (entier et décimaux).

Alors que parseInt() le fait seulement vers des chiffres entiers.
