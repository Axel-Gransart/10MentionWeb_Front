Les string en JavaScript

Nous pouvons initialiser une chaine de caratère de plusieurs façon

Avec des guillemets simples (simple quotes) :

  const PRENOM = 'John';

Avec des guillemets doubles (double quotes) :

  const NOM = "DOE";

Avec les simples quotes et un backslash (ou antislash) : Permet d'échapper un caractère dans la string

  const ADRESSE = 'J\'habite à NewYork';
  // On peut aussi noter "J'habite à NewYork"
  document.write("Je m'appelle " + PRENOM + " " + NOM + ", " + ADRESSE);

Modification d'un type -> string en type -> number

  let chiffre = "12.5";
  console.log(typeof chiffre);
  console.log(chiffre);
      
  let chiffreDecimal = parseFloat(chiffre);
  console.log(typeof chiffreDecimal);
  console.log(chiffreDecimal);
      
  let chiffreEntier = parseInt(chiffre);
  console.log(typeof chiffreEntier);
  console.log(chiffreEntier);
