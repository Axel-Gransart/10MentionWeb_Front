Les boucles en JavaScript

Les boucles nous permettent d'exécuter une ou plusieurs instructions un certain nombre de fois

  • - Une boucle respecte une condition
  • - Tant que cette condition est respectée, nous allons exécuter des instructions en boucle.
  • - La plupart du temps, nous utilisons les boucles pour parcourir chaque élément d'un tableau
  • - Il y a différents types de boucles, les plus utilisées sont la boucle "FOR" et la boucle "WHILE"

Les boucles FOR

SYNTAXE :

    
  for( 3 arguments : initialisation; condition de la continuité; ensuite l'incrémentation) {    
    code à executer
  }

Exemple :

  for (let i = 0; i < 10; i++ ) {
    document.write(`i est égal à : ${i}`);      
  }

- i = 0 j'initialise le départ de la boucle à 0
- i < 10 tant que la valeur de i est inférieure à 10
- i++ j'incrémente un tour de boucle

En gros je donne l'instruction de faire un tour de boucle supplémentaire tant que la valeur de i n'est pas égale ou supérieure à 10

Je boucle 2 par 2

  for (let j = 0; j < 10; j += 2) {
    document.write(`j est égal à : ${j}`);      
  }

Boucle WHILE (tant que)

SYNTAXE

  initialisation de la variable

  while(expression qui vaut True ou False) {        
    code à executer tant que l'expression vaut True
    Incrémentation de la variable
  }

Exemple :

  let k = 0; // J'initialise une variable k égale à 0
  
  while (k <= 5) { // Tant que k est inférieur ou égale à 5
  document.write(`k est égale à ${k} / `); //  J'exécute l'instruction
  k++ // A chaque tour de boucle, j'incrémente la variable
  };

Intégration d'une condition if - else dans une boucle WHILE

  let l = 0;

  while (l <= 5) {  
    if (l == 5) {
      document.write(`l est égale à ${l} <- `);
    }
    else {
      document.write(`l est égale à ${l} / `);
    }
    k++  
  };

Au vu des conditions inscrites dans la boucle ci-dessus, Nous avons un '/' à la fin des 5 premiers tours et un '<-' à la fin du sixième tour.
