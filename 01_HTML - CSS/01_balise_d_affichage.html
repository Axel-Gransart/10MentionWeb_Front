le CSS: Le Cascading Style Sheet

Le CSS permet de faire du style sur nos pages html

CSS dans la balise

La première façon de faire du style dans un élément est de déclarer l'attribut style sur celui-là. Ce n'est pas la meilleure manière de le faire

présentation 1ère façon de style

CSS dans l'en-tête

Une seconde façon de faire du style sur notre page est de mettre la balise style dans la partie head. Cette methode n'est toujours pas la plus conseillée mais elle permet plus de chose que la méthode précédente. On peut desormais grâce à cette méthode viser plusieurs éléments d'un coup. Exemple, on met du gris en fond sur le corps de la page, les h2 en rouge et les h3 avec un fond rose

Le CSS dans une feuille de style externe

La dernière et meilleure façon pour faire du style est la méthode suivante: on crée un dossier CSS avec un fichier style.css à l'intérieur. Il faudra alors lier nos 2 fichiers grâce à la balise link. On pourra alors faire du style sur les éléments qui se trouvent dans notre page html. exemple, on déclare que tout les articles doivent être avec une bordure violette
