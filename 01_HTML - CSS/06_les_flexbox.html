C - direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction, on peut les positionner verticalement ou encore les inverser. Il peut prendre les valeurs suivantes:
-> row: organisés sur une ligne (par défaut)
-> column: organisés sur une colonne
-> row-reverse: organisés sur une ligne mais en ordre inversé

D - Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1- Alignement sur l'axe principal

Afin d'aligner nos éléments sur l'axe principal: on utilise la propriété du flex justify-content

Cette propriété gère la distribution des emplacements des flex-items suivant la place disponible dans le flex-container. Elle le fait selon le positionnement de l'axe principal.

  • - flex-start: les éléments sont alignés au début (valeur par défaut)
  • - flex-end: les éléments sont alignés à la fin
  • - center: les éléments sont alignés au centre
  • - space-between: les éléments sont étirés sur tout l'axe du conteneur (espace entre)
  • - space-around: les éléments sont étirés sur tout l'axe du conteneur mais laisse un espace aux extrémités (espace autour)
  • - space-evenly: Les éléments sont répartis uniformément dans le conteneur le long de l'axe principal

E - retour à la ligne

Par défaut, les blocs essaient de rester sur la même ligne, même s'ils n'ont pas la place (ce qui provoque parfois des problèmes de conception). Si vous le souhaitez, vous pouvez demander au block de passer à la ligne lorsqu'ils n'ont plus de place. Pour cela, vous devez utiliser la propriété Flex-wrap.

Elle peut prendre les valeurs suivantes

  • - nowrap: Pas de retour à la ligne (valeur par défaut)
  • - wrap: Les éléments vont à la ligne lorsqu'il n'y a plus de place
  • - wrap-reverse: Les éléments vont à la ligne lorsqu'il n'y a plus de place mais en sens inverse

F - Répartir plusieurs lignes

Si vous avez plusieurs ligne dans votre flexbox,vous pouvez choisir comment celles-ci seront réparties avecalign-content. Cette propriété permet de gérer l'alignement des différents lignes (ou colonnes) les par rapports au autres. Cette propriété n'aura d'éffet que si le conteneur flexible posséde plusieurs lignes(si l'axe principale est horizontal) ou plusieurs colonnes ( si l'axe principle est vertical) et que si le flex-wrap est présent.

Les valeurs de cette propriété :


  • - flex-start: les éléments sont placés au début
  • - flex-end: les élélements sont placés à la fin
  • - center: les éléments sont placés au centre
  • - space-between: les éléments sont séparés avec de l'éspace entre eux
  • - space-around: les éléments sont séparés avec de l'éspace entre eux, mais il y'a aussi de l'éspace au début et à la fin
  • space-evenly : Les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal.
  • - stretch: les éléments s'étirent pour occuper tout l'éspace (valeur par défaut)
